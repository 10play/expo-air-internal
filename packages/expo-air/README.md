# expo-air

This is an alias package for [`@10play/expo-air`](https://www.npmjs.com/package/@10play/expo-air).

## Usage

```bash
# These are equivalent:
npx expo-air init
npx @10play/expo-air init

npx expo-air fly
npx @10play/expo-air fly
```

## Installation

For full documentation and installation instructions, see the main package: [@10play/expo-air](https://github.com/10play/expo-air)
